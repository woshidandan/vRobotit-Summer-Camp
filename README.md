## 其它注意事项
- 🤖是否允许使用大模型完成作业：允许，但请在最终答辩时表明哪些内容由大模型完成，诚信第一哦！
- 🎤答辩时主要看中什么：夏令营作业的完成情况📝、自主性和创新潜力💡、语言表达的流畅度🗣️、PPT的美观度🎨、未来规划与实验室发展的契合度🤝。
- 😢特别想要进组，但最后落选怎么办：别灰心，可以选择直博🎓，直博不占用名额，我们也欢迎同学直博，一起探索科研的奥秘吧！
